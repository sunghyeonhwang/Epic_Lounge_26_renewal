# 0209 수정사항 
A. 공통
1. 트위터 공유하기 버튼 X 공유하기 버튼으로 교체

B. 새소식
1. 새소식 검색영역 작동 확인 - "언리얼"로 검색시 검색결과 없음 
2. 새소식 검색창에 버튼 디자인 width값 넓게 

C. 이벤트 
1. 이벤트 페이지 상단 class="v4-search-tabs" 영역에 "전체 진행중 종료 결과발표" 탭 인터페이스 생성 가운데 정렬

D. 리소스 
레거시와 비교해서 인터페이스/디자인는 동일하게 맞춰줘야함 목록 / 보기등이 기존과 차이가 많음, 기능적인것 외에도 네비바가 가린다던지 하는 경우도 있음
1. 필터란에 검색은 삭제 
2. 레거시와 비교시에 인터페이스가 너무 다름, 확인해서 전체 수정 필요 (특히 필터)
3. 목록에서 각 게시물 이미지 썸네일이 잘못되어 있음 
4. 무료 콘텐츠는 클릭하면 새창 연결임 
5. 게시물 클릭시에 내용이 다 안보이고 잘리는 (네비바에 의해)
6. 레거시와 비교해서 게시물 읽기의 html로 잡아놓은 테이블들이 적용이 안된경우가 있음
